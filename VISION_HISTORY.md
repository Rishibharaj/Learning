# Vision History

Tracks how the project's vision was defined and how it changes over time. Most projects' vision drifts silently; this file exists so that any drift is visible and intentional.

## v1 — Sprint 0 (initial vision)

Knowledge Explorer is a domain-independent interactive learning platform. The application itself carries no subject-specific knowledge; all of that is supplied through "Knowledge Packs." AI is the first pack, built from an existing Plotly sunburst prototype. PMP is the second, planned explicitly to prove the architecture generalizes. Further packs (Cloud Computing, Python, Networking, DevOps, Data Science, Cyber Security, Enterprise Automation) are named as future targets but not scoped in detail yet.

The long-term goal is a portfolio-quality, open-source project demonstrating product thinking, software architecture, visualization, AI knowledge, PMP knowledge, clean UI, and professional documentation — published to GitHub and LinkedIn.

**Status as of Sprint 1: unchanged.** Sprint 1 confirmed rather than revised this vision — the work done (navigation, search, theming, breadcrumb, dynamic cards) was entirely in service of making the existing vision's first pack feel professional, with no scope changes to the vision itself.
